# Responsive Website Mini Project

A responsive and user-friendly website developed as a **Basic Web Design Mini Project** using **HTML, CSS, JavaScript, and Bootstrap**.

## 📌 Overview
This project demonstrates the implementation of **responsive web design principles** to create a website that adapts smoothly across different screen sizes and devices.

## ✨ Features
- Fully responsive layout
- Multiple pages
- Modern and clean design
- Bootstrap components
- Mobile-friendly interface
- Structured navigation

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap

## 📂 Pages Included
- Home
- About
- Events
- Certificate
- Contact
